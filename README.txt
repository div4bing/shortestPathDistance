1. Compile:
  To Compile this binary, run "make"

2. Clean:
  To clean the binary and .o files, run "make clean"

3. Run:
  To run the program, please follow below steps:
  ./submission input.txt output.txt
  input.txt should have the path from 1 node to other with distance for ex. 1 2 140
  output.txt will have all node distances with shortest distance and it's shortest path, for ex. 1 2 140 path:1->2

4. Test with other City Maps:
  If required to test this program with other city maps, please modify the "NUMBER_OF_CITY" macro to number of city/node for new map and
  input respective data through input.txt file
